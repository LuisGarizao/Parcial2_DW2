🌤️ Fake Weather API

Bienvenido a la API de Clima Falso. Endpoints disponibles:

  • 🌎 Ver todos los reportes

    -> /reports/all

  • 🌎 Ver todas las ciudades

    -> /cities/all

  • 🏙️ Ver reportes de una ciudad (ejemplo: Bogota)

    -> /reports/search

  • 🏙️ Ver si una ciudad está en nuestros registros (ejemplo: Tokyo)

    -> /cities/search
